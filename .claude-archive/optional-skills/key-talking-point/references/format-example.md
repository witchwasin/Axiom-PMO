### <span style="color:#1A237E">**สไลด์ 3 — Key Concepts `[5 นาที]`**</span>

เปลี่ยนสไลด์

<div style="border-left: 4px solid #FFB300; background-color: #FFF8E1; padding: 10px; margin: 10px 0;">

**คำศัพท์สำคัญ:**
- **Visibility** <span style="color:#C62828">**(วิสิบิลิตี้)**</span> — ความสามารถในการมองเห็นข้อมูลทั้งห่วงโซ่
- **Traceability** <span style="color:#C62828">**(เทรซะบิลิตี้)**</span> — ความสามารถในการติดตามย้อนกลับ
- **Resilience** <span style="color:#C62828">**(รีซิลเลียนซ์)**</span> — ความยืดหยุ่นทนทานต่อวิกฤต

</div>

<span style="color:#1565C0">**ก่อนเข้าเนื้อหา มาทำความรู้จักคำศัพท์ 3 คำที่จะได้ยินบ่อยมากวันนี้ครับ**</span>

<span style="color:#1565C0">**คำแรก Visibility**</span> <span style="color:#C62828">**(วิสิบิลิตี้)**</span> <span style="color:#1565C0">**แปลตรงตัวว่า "การมองเห็น" ในบริบท Supply Chain หมายถึง การเห็นว่าสินค้าอยู่ตรงไหน สถานะเป็นอย่างไร ทุกจุดตลอดเส้นทาง เหมือนเราเปิด Google Maps แล้วเห็นรถส่งของวิ่งอยู่ตรงไหน**</span>

<span style="color:#1565C0">**คำที่สอง Traceability**</span> <span style="color:#C62828">**(เทรซะบิลิตี้)**</span> <span style="color:#1565C0">**คือการติดตามย้อนกลับได้ สินค้าชิ้นนี้มาจากไหน ผลิตเมื่อไหร่ ผ่านมือใครมาบ้าง ถ้ามีปัญหา ตามกลับไปถึงต้นตอได้ทันที**</span>

Check-in: "สองคำนี้ต่างกันตรงที่ Visibility คือเห็นปัจจุบัน Traceability คือตามย้อนอดีต จำง่ายๆ แบบนี้นะครับ"

<span style="color:#1565C0">**คำสุดท้าย Resilience**</span> <span style="color:#C62828">**(รีซิลเลียนซ์)**</span> <span style="color:#1565C0">**แปลว่าความยืดหยุ่น ทนทาน Supply Chain ที่มี Resilience คือเจอปัญหาแล้วฟื้นตัวเร็ว ปรับตัวได้ ไม่ล้มทั้งระบบ**</span>

> 💡 *เกริ่นคำศัพท์ก่อนเข้าเนื้อหา ช่วยให้ผู้ฟังไม่สะดุดเวลาเจอคำเหล่านี้ซ้ำในภายหลัง*
