# pmo-delivery

Use this skill for work item planning, delivery handoff, task source-of-truth management, and engineering sequencing.

Required outputs:

- `DELIVERY.md` work items or GitHub Issue references
- Mode, strict trigger, mode reason, and mode approval per work item
- Requirement and design references
- Acceptance criteria
- Test checklist
- Owner, status, review stage, and evidence reference

Use one task source of truth only.
