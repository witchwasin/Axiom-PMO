# pmo-intake

Use this skill for intake, source review, requirement extraction, gap analysis, and scope clarification.

Read only the minimum source set defined in `CONTEXT-ROUTER.md` and `pmo-config/context-map.yaml`.

Required outputs:

- Confirmed requirements with `source_ref`
- Assumptions
- Open questions
- Source Snapshot updates for `PROJECT.md`
- Recommended mode per work item when risk is visible

Never invent requirement details not found in source.
